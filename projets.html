Projet 1 : Application pour entreprise de mutuelle

Description : Développement d'une application qui permet aux commercials de l'entreprise de créer une offre spcécifiques pour son client. Avec une interface ergonimique et pratique. Exemple : possibilité de création de powerpoint pour le client...

Langages utilisés : JavaScript, HTML, CSS, PHP, SQL

Framework : PHP LARAVEL

Évolutions effectuées 1 : Création d'un système qui permet la génération de fichiers CSV rempli automatiquement avec les différents code défini par l'application. Cela permet d'obtenir quel types de prestations et le plus utilisés et surtout cela permet de faire des récapitulatifs pour l'entreprise sur ses prestations les plus vendus.

Évolutions effectuées 2 : Modifications d'un services en backend. Le transfert d'archive se faisait uniquement par SFTP (user+ password). Il se fait désormais avec une clé SSH plus une passphrase sur la clé ainsi que user plus mot de passe.
